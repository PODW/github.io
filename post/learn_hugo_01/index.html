Hugo 學習日誌01

 2020-12-29 Hugo 網站架設 

本站是透過Hugo這個靜態網站生成器製作而成的
網路上還有其他的靜態網站生成器
而我會選擇用Hugo的原因就是因為它安裝簡單生成快速
我這裡只記錄怎麼用Hugo製作靜態網站跟我遇到的問題
其他更詳細的內容請再上網搜尋

(一)安裝 Hugo

說是安裝Hugo其實只需要一個hugo.exe就可以執行
所以只要去  Hugo官網下載  符合你電腦環境的執行檔就好

我是在Windows10 x64系統環境下所以是載

hugo_extended_(0.79.0)1_Windows-64bit.zip

將檔案解壓縮後就會得到一個 hugo.exe
其實現在就可以執行Hugo了，但是只能在目前解壓縮出來的目錄下
為了之後執行方便我建議把修改環境變數 2
讓Windows PowerShell或Cmd不管在哪都可執行Hugo

照著圖片操作最後按下確認就完成Hugo的安裝
這時候可以打開Cmd來確認，執行hugo  version
可以看到版號就代表成功了

(二)創建一網站

hugo server -> 運行本機的server來顯示目前的網頁

hugo new site 名稱 -> 創建一個新的網站

hugo new post/文章標題.md -> 創建一個新的文章

hugo -> 將目前製作的網頁輸出成 html

  1. 看目前Hugo最新是哪一版 ↩︎

  2. (Windows)作業系統會依照PATH環境變數中所設定的路徑順序，依序尋找各路徑可以執行命令的執行檔 ↩︎
